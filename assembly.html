x86 Assembly

x86 Assembly is a low-level programming language which can be used for many operations. This programming language is heavily centered around registers which allow you to do operations in Assembly programs. There are ten 32-bit and six 16-bit processor registers in IA-32 architecture. The registers are grouped into three categories which include, General registers, Control registers, and Segment registers. General registers are divided into the following groups, Data registers, Pointer registers, and Index registers. If wanting to know more that isn't in this brief description, click here.

Data Registers:

Four 32-bit data registers are used for arithmetic, logical, and other operations. These 32-bit registers can be used in three ways -

As complete 32-bit data registers:
EAX, EBX, ECX, & EDX.

Lower Halves of the 32-bit registers can be used as four 16-bit data registers:
AX, BX, CX, & DX.

Lower and higher halves of the above-mentioned four 16-bit registers can be used as eight 8-bit data registers:
AH, AL, BH, BL, CH, CL, DH, & DL.
